Como surgiu o HTML

inicio

O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente. Ele publicou a primeira versão do HTML em 1991, consistindo em 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos (modificadores de tags). Devido a rápida ascensão e popularidade, o HTML é agora considerado um padrão oficial da web. O maior upgrade da linguagem foi o lançamento do HTML5 em 2014. Diversas novas tags semânticas foram adicionadas que revelam o significado do seu próprio conteúdo, como.


Quem é Tim Berners-Lee?

inicio

Timothy John Berners-Lee KBE, OM, FRS (TimBL ou TBL). Nascido em Londres, no dia 8 de junho de 1955 é um físico britânico, cientista da computação e professor do MIT. É o criador da World Wide Web, tendo feito a primeira proposta para sua criação a 12 de março de 1989. Em 25 de dezembro de 1990, com a ajuda de Robert Cailliau e um jovem estudante do CERN, implementou a primeira comunicação bem-sucedida entre um cliente HTTP e o servidor através da internet.

Tags HTML

inicio

Atualmente, existem 142* tags HTML disponíveis, que permitem a criação de diversos elementos. Mesmo que os navegadores modernos não suportem algumas dessas tags, podemos nos beneficiar de aprender sobre os diferentes elementos disponíveis.


Algumas tags mais utilizadas no HTML


Tag HTML

inicio

É o elemento raiz que define todo o documento HTML.


Tag head

inicio

Responsável pelos os metadados, titulo da página e a codificação de caracteres da página.


Tag body

inicio

Responsável por envolver todo o contéudo mostrado na página.


Tag h1...h6

inicio

Variam entre h1 e h6, onde h1 tem o tamanho maior, que vai diminuindo até chegar no h6.


Tag p

inicio

Tag responsável pela criação de parágrafos, são abertas e fechadas usando a tag p


Tag ol

inicio

Tag que dá origem a uma lista ordenada.


Tag ul

inicio

Tag que dá origem a uma lista não ordenada.


Tag li

inicio

Esta tag siginifica list item ou item da lista em português, utilizada para inserir um item na lista, deve ser colocada entre as tags ul ou ol.


Tag strong

inicio

Esta tag faz o elemento envolvido por elas ser renderizado em negrito, desta forma.


Tag i

inicio

Esta tag faz o elemento aparecer em itálico.


Tag a

inicio

Esta tag quanto envolve um elemento ela adiciona um hiperlink que tem sua direção definida pelo atributo href.

Confira o exemplo abaixo!.:
